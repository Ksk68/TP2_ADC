# 🗓️ Sistema de Reservas - TP2_ADC

[![Python Version](https://img.shields.io/badge/python-3.13-blue.svg)](https://www.python.org/downloads/release/python-3130/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Sistema robusto para gestão de marcações entre Clientes e Estabelecimentos, desenvolvido como projeto prático para a unidade curricular de **Ambientes de Desenvolvimento Colaborativo**.

---

## 🎯 Objetivo do Projeto
O foco desta aplicação é permitir que utilizadores realizem agendamentos e gerenciem o seu perfil de forma centralizada, enquanto os proprietários de negócios podem atualizar as informações dos seus estabelecimentos em tempo real.

## ✨ Funcionalidades Principais

### Para Clientes
* **Autenticação:** Login seguro através de **Nome de Utilizador** e palavra-passe.
* **Área do Cliente:** Dashboard dedicado para visualizar e gerir o histórico de marcações.
* **Cancelamento:** Opção para remover ou cancelar marcações caso o utilizador já não possa comparecer.

### Para Donos de Estabelecimento
* **Gestão de Negócio:** Criação do registo do negócio e edição de dados (nome, morada, telefone).
* **Validação de Dados:** Verificação de horários de funcionamento para garantir que as informações estão sempre corretas e formatadas.

---
